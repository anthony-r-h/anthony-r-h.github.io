# About

The DOXA 300T is the older reference in this line. It came out of the classic SUB platform and set the core format: no-nonsense dive case, high legibility, and the decompression-style bezel language DOXA is known for.

![](/files/SEESTERN600T/1/post/c1.png)

The 600T came later with a different case direction. It is more angular and heavier in shape, with a more geometric profile than the 300T case. Same family, different stance.

![](/files/SEESTERN600T/1/post/c3.png)

# Build

This one uses the Seestern 600T case as the base and mixes it with parts that push it closer to the look I wanted.

- Base watch: Seestern 600T
- Dial: DOXA 300T inspired from Yobokies
- Hands: Ploprof inspired from Yobokies
- Bracelet: Strapcode

The end result is a 600T-shaped build with 300T influence in the dial direction, plus hand set changes to finish it.
